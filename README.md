# - 전자 액자 만들기 
사용언어 - Kotlin 

원하는 사진 최대 6개까지 선택해서 전자액자로 나타내줍니다.

갤러리 권한을 받아옵니다.


타이머로 5초마다 사진이 전환됩니다.


* 사용된 중요한 것 들
1. 권한 허락
2. 타이머
3. onStop()
4. onStart()
5. onDestory()	
6. isNotEmpty()
7. forEach()
8. forEachIndex()
9. runOnUiThread()
10. setImageURI()
11. intent
12. getIntExtra()
13. getStringExtra()
14. AlertDialog.builder()
15. and so on
